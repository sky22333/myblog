- [x] react
- [x] ShadcnUI
- [x] Tailwind CSS


## 功能实现

- [x] 实现深色/浅色模式
- [x] 流畅的动画效果
- [x] 响应式设计
- [x] 良好的SEO和性能
- [x] 优化图片处理
- [x] Markdown编写

- [x] 站内文章搜索
- [x] 图片灯箱
- [x] 回到顶部

## 站点配置

- 站点配置：`src/content/config/site.json`
- 域名配置：`astro.config.mjs`

`src/content/authors/default.json`文件配置：当文章引用"default"作为文章作者或没有明确指定作者时，系统会使用这个文件中的信息。



## 构建
```
pnpm i
```
```
pnpm run build
```
